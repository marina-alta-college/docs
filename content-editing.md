# Content Editing Guide

This guide is for non-technical users who need to edit website content.

## Accessing the CMS

1. Go to https://marina-alta-college.sanity.studio
2. Log in with your Sanity account
3. You'll see the content sidebar on the left

## Editing a Page

The sidebar shows all pages:

- **Home Page** — hero section, fast facts strip, content sections
- **What We Offer** — educational programmes and philosophy
- **Who We Are** — team bios, advisors, mentors
- **Admissions & Fees** — admissions process, fees, scholarships
- **Contact Page** — contact information and intro text
- **Site Settings** — site name, tagline, email, phone, address

Click any page to edit it. Make your changes and click **Publish** (green button, bottom right).

Changes appear on the live website within a few seconds of publishing.

## Writing in Multiple Languages

Every text field appears four times:

- **Title (EN)** — English (required)
- **Title (ES)** — Spanish (optional)
- **Title (DE)** — German (optional)
- **Title (NL)** — Dutch (optional)

If you leave a translation empty, the website shows the English version as a fallback.

## Adding a Journal Post

1. Click **Journal** in the sidebar
2. Click the **+** button (top right)
3. Fill in:
   - **Title** — the headline
   - **Slug** — click "Generate" to create from title (this becomes the URL)
   - **Excerpt** — short summary shown on the listing page
   - **Body** — the full article (use the rich text editor)
   - **Cover Image** — optional header image
   - **Author** — author name
   - **Published At** — date and time
4. Click **Publish**

## Adding a Calendar Event

1. Click **Calendar** in the sidebar
2. Click the **+** button
3. Fill in:
   - **Title** — event name
   - **Event Type** — Term, Holiday/Break, Event, or Exam
   - **Start Date** — when it starts
   - **End Date** — when it ends (leave empty for single-day events)
   - **Description** — optional details
4. Click **Publish**

## Editing Site Settings

Site Settings controls global information used across the whole site:

- **Site Name** — appears in the browser tab and footer
- **Tagline** — appears in the footer and SEO
- **Contact Email** — shown on the contact page and footer
- **Contact Phone** — shown on the contact page
- **WhatsApp Number** — powers the floating WhatsApp button
- **Address** — shown in the footer and contact page

## Images

- Use high-quality images (the site optimises them automatically via Sanity CDN)
- Always fill in the **Alt Text** field — this is important for accessibility
- The hotspot tool lets you choose which part of the image to focus on when it's cropped

## Tips

- You can **undo** changes before publishing using Ctrl+Z / Cmd+Z
- The **Review Changes** button shows what you've changed before publishing
- If you make a mistake, you can always edit and re-publish
- Draft changes (unpublished) are only visible in the Studio, not on the live site
